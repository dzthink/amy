# amy

Personal agent app scaffold.

- Web client: `web`
- Agent core (ADK): `agent`
- Markdown data store: `data`
- Product docs: `docs`

Quick start:
- `cd agent && uv venv && source .venv/bin/activate && uv pip install -e . && python -m agent_core.server`
- `cd web && npm install && npm run dev`
